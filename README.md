# only-enes-no-ai-
A repository for me to create some projects without any generative ai usage to sharpen my skills. There is no AI used in this repository including Google's AI Overview.

I am aware that coding with AI is the future, and I am not against it. However, as a student, I need to be able to rely myself as much as I rely on AI. So, even though I will continue using AI for my major projects that needs faster deployments, I will not use any generative AI here. 

Projects here will be mostly dumb, meaningless, no-purpose projects. Similar to what old computer science students were building on their free time before AI.
