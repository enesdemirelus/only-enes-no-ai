A repository for me to create some projects without any generative AI usage to sharpen my skills. There is no AI used in this repository, including Google's AI Overview.

I am aware that coding with AI is the future, and I am not against it. However, as a student, I need to be able to rely on myself as much as I rely on AI. So, even though I will continue using AI for my major projects that need faster deployments, I will not use any generative AI here.

Projects here will be mostly dumb, meaningless, no-purpose projects. Similar to what old computer science students were building in their free time before AI.

To be able to achieve this, I will try not to use any IDE as much as possible. I will be using NeoVim as my text editor.
